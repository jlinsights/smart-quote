## 1. 시작하기

Smart Quote System은 한국 발 국제 택배/화물의 견적을 빠르게 산출하는 내부용 도구입니다. 별도의 로그인 없이 웹 브라우저에서 바로 사용할 수 있습니다.

### 지원 운송사

| 운송사       | 대상 국가     | 비고              |
| --------- | --------- | --------------- |
| **UPS**   | 전 세계 22개국 | 기본 운송사          |
| **DHL**   | 전 세계 22개국 |                 |
| **E-MAX** | 중국, 베트남만  | 해당 국가 선택 시만 활성화 |

### 지원 목적지 국가 (22개국)

- **아시아-태평양**: 중국, 일본, 베트남, 싱가포르, 홍콩, 대만, 태국, 필리핀, 호주, 인도
- **미주**: 미국, 캐나다, 브라질
- **유럽**: 독일, 영국, 프랑스, 이탈리아, 스페인, 네덜란드
- **중동**: UAE, 사우디아라비아, 터키

## 2. 화면 구성

### 상단 헤더

화면 최상단에 항상 표시되며, 아래 기능 버튼들이 있습니다.

| 버튼                         | 위치   | 기능                      |
| -------------------------- | ---- | ----------------------- |
| **Calculator / History 탭** | 중앙   | 견적 계산기와 이력 화면 전환        |
| **견적 초기화** (회전 화살표)        | 우측   | 입력값 전체 초기화 (확인 대화상자 표시) |

## 3. 견적 계산하기

입력값을 변경하면 **자동으로 즉시 계산**됩니다. 별도의 "계산" 버튼은 없습니다.

### 3.1 노선 및 조건 (Route & Terms)

#### 출발지
- 한국(KR) 고정

#### 도착 국가
- 드롭다운에서 22개국 중 선택

#### 도착지 우편번호
- 국가별로 입력 형식이 자동 안내됩니다
  - 미국: `90001` (5자리)
  - 일본: `100-0001` (우편번호 7자리)
  - 영국: `SW1A 1AA`
  - 기타: 국가별 형식

#### 운송사 선택
- **UPS**: 모든 국가 이용 가능 (기본값)
- **DHL**: 모든 국가 이용 가능
- **E-MAX**: 중국/베트남만 선택 가능 (다른 국가에서는 비활성화)

#### 인코텀즈 (Incoterms)
무역 조건을 선택합니다. 선택에 따라 견적 범위가 달라집니다.

| 인코텀즈    | 설명       | 견적에 포함되는 비용         |
| ------- | -------- | ------------------- |
| **EXW** | 공장 인도    | 포장비만 (국제운송 제외)      |
| **FOB** | 본선 인도    | 포장비만 (국제운송 제외)      |
| **C&F** | 운임 포함    | 포장비 + 국제운송비         |
| **CIF** | 운임+보험 포함 | 포장비 + 국제운송비         |
| **DAP** | 도착지 인도   | 포장비 + 국제운송비 (관세 제외) |
| **DDP** | 관세 포함 인도 | 포장비 + 국제운송비 + 관세/세금 |

> **참고**: EXW, FOB 선택 시 "Collect Term" 경고가 표시됩니다. 국제운송비가 견적에서 제외되므로 주의하세요.

---

### 3.2 화물 정보 (Cargo Details)

#### 단위 전환
- 우측 상단의 **cm/kg** 또는 **in/lb** 토글로 미터법/야드파운드법 전환 가능
- 내부 계산은 항상 cm/kg 기준으로 수행됩니다

#### 박스 추가/관리

| 동작    | 방법                                      |
| ----- | --------------------------------------- |
| 박스 추가 | **+ Add Box** 버튼 (새 박스는 마지막 박스의 치수를 복사) |
| 박스 복제 | 각 박스 우측의 **복사 아이콘**                     |
| 박스 삭제 | 각 박스 우측의 **휴지통 아이콘** (최소 1개는 유지)        |

#### 각 박스 입력 항목

| 항목              | 설명          | 비고            |
| --------------- | ----------- | ------------- |
| **Qty** (수량)    | 동일 규격 박스 수량 | 정수만 입력        |
| **L** (가로)      | 가장 긴 변 (cm) | 122cm 초과 시 경고 |
| **W** (세로)      | cm          |               |
| **H** (높이)      | cm          |               |
| **Weight** (중량) | 실중량 (kg)    | 25kg 초과 시 경고  |

#### 자동 경고 표시

박스 입력 시 아래 경고가 자동으로 표시됩니다:

- **중량 \> 25kg**: "may need surge fee" - AHS(Additional Handling Surcharge) 대상
- **가로 \> 122cm**: "check AHS Dim" - 대형 화물 할증 대상
- **둘레 \> 300cm**: 대형 패키지 할증 대상
- **가로 \> 274cm 또는 둘레 \> 400cm**: UPS 최대 규격 초과

### 3.3 부가 서비스 (Value Added Services)

| 항목                  | 설명                                                            |
| ------------------- | ------------------------------------------------------------- |
| **Special Packing** | `NONE`(없음) / `WOODEN_BOX`(목상자) / `SKID`(스키드) / `VACUUM`(진공포장) |
| **포장비 수동 입력**       | 비워두면 자동 계산. 금액을 입력하면 자동 계산을 무시하고 입력값 사용                       |
| **Surge / AHS Fee** | 운송사 할증료(AHS, 대형화물, 성수기 등)를 수동 입력                              |
| **관세/세금 (DDP만)**    | 인코텀즈가 DDP일 때만 표시. 도착지 예상 관세 입력                                |

> **포장비 자동 계산 기준**:
> - 자재비: 화물 표면적(m2) x 15,000원
> - 인건비: 박스당 50,000원 (진공포장 시 1.5배)
> - 훈증비: 30,000원 (목상자/스키드 포장 시 고정)
> - 핸들링: 35,000원 (수출통관/서류 고정)

### 3.4 금융 조건 (Financial Factors)

| 항목                      | 기본값   | 설명                                |
| ----------------------- | ----- | --------------------------------- |
| **환율 (KRW/USD)**        | 1,300 | 원/달러 환율. 참고 링크: Naver 환율          |
| **FSC %**               | 30%   | 유류할증료 비율. 참고 링크: UPS FSC, DHL FSC |

## 4. 견적 결과 확인

입력값 변경 시 우측(또는 하단) 결과 패널에 **실시간으로** 계산 결과가 표시됩니다.

### 4.1 총 견적 금액 (Quote Summary)

- **KRW 금액**: 큰 글씨로 표시 (예: ₩1,234,500)
- **USD 환산**: 아래에 참고 표시 (예: ≈ $950.38 USD)
- **운송 정보**: 예상 소요일, 적용 Zone

### 4.2 주요 지표 (Key Metrics)

| 지표               | 설명                              |
| ---------------- | ------------------------------- |
| **Act. Weight**  | 실중량 합계 (kg)                     |
| **Bill. Weight** | 청구중량 — 실중량과 부피중량 중 큰 값          |
| **Margin**       | 이익률 (%). 10% 미만이면 빨간색, 이상이면 초록색 |

> **부피중량 계산**:
> - UPS/DHL: 가로 x 세로 x 높이 / 5,000
> - E-MAX: 가로 x 세로 x 높이 / 6,000

### 4.3 원가 내역 (Cost Breakdown)

"Internal" 라벨이 표시된 내부용 원가 분석 카드입니다.

#### 원가 항목

| 항목                      | 설명                                         |
| ----------------------- | ------------------------------------------ |
| **Packing & Docs**      | 포장비 합계 (자재+인건+훈증+핸들링)                      |
| **Intl. Freight**       | 국제운송비 (Base Rate + FSC + War Risk + Surge) |
| **Dest. Duty**          | 도착지 관세 (DDP일 때만)                           |
| **Total Internal Cost** | 총 원가                                       |

#### 가격 전략

| 항목                    | 설명                                  |
| --------------------- | ----------------------------------- |
| **Margin**            | 이익 금액과 비율. 이 필드에서 직접 마진을 수정할 수 있습니다 |
| **Final Quote Price** | 최종 견적가 (원가 + 마진, 100원 단위 올림)        |

> **직접 수정 가능한 필드**: 포장비와 마진(USD)은 원가 내역 카드에서 **클릭하여 직접 수정**할 수 있습니다. 수정 즉시 최종 견적가에 반영됩니다.

### 4.4 경고 알림 (Warnings)

아래와 같은 상황에서 노란색 경고 배너가 표시됩니다:

- 이익률이 10% 미만일 때
- 부피중량이 실중량보다 클 때
- Surge/AHS 요금이 적용될 때
- EXW/FOB (Collect Term) 조건일 때

## 5. 견적 저장하기

계산 결과가 있을 때 상단의 **Save Quote** 버튼으로 견적을 서버에 저장할 수 있습니다.

### 저장 절차

1. **Save Quote** 버튼 클릭
2. 메모 입력란 표시 (선택사항 - 비워도 됨)
3. **Save** 클릭 또는 Enter 키
4. 저장 성공 시: 초록색으로 **"Saved! SQ-2025-XXXX"** 표시
5. **View** 링크 클릭 시 이력 화면으로 이동

### 참고 사항

- 메모는 선택사항입니다. 비워두고 저장해도 됩니다
- 동일 조건으로 이미 저장된 견적이 있으면 중복 저장 확인 대화상자가 표시됩니다
- 저장된 견적에는 자동으로 `SQ-YYYY-NNNN` 형식의 참조번호가 부여됩니다
- 저장에 실패하면 빨간색 "Failed to save" 메시지가 3초간 표시됩니다

## 7. 견적 이력 관리

상단 **History** 탭을 클릭하면 저장된 모든 견적을 조회할 수 있습니다.

### 7.1 검색 및 필터

#### 텍스트 검색
- 검색창에 참조번호(SQ-XXXX) 또는 도착지를 입력 후 **Search** 클릭

#### 상태 필터
**Filters** 버튼을 클릭하면 상태별 필터가 표시됩니다:

| 상태           | 색상  | 의미         |
| ------------ | --- | ---------- |
| **draft**    | 회색  | 초안 (기본 상태) |
| **sent**     | 파란색 | 고객에게 전송됨   |
| **accepted** | 초록색 | 고객 수락      |
| **rejected** | 빨간색 | 고객 거절      |

- 상태 버튼을 클릭하면 해당 상태만 필터링
- 같은 버튼을 다시 클릭하면 필터 해제
- **Clear all**로 모든 필터 초기화

### 7.2 이력 목록

| 열            | 내용                       |
| ------------ | ------------------------ |
| Ref No       | 견적 참조번호                  |
| Date         | 생성일                      |
| Dest         | 도착 국가 코드                 |
| Amount (KRW) | 총 견적 금액 (원화)             |
| USD          | 달러 환산                    |
| Margin       | 이익률 (10% 이상: 초록, 미만: 주황) |
| Status       | 상태 배지                    |
| Actions      | 상세 보기 / 삭제 버튼            |

- 페이지당 20건 표시
- 하단 페이지네이션으로 이동

### 7.3 상세 보기

목록에서 **눈 모양 아이콘**을 클릭하면 견적 상세 모달이 열립니다.

모달에서 확인할 수 있는 정보:
- 주요 지표 (금액, USD, 마진, 청구중량)
- 노선 및 서비스 정보
- 화물 항목 상세
- 원가 내역 (항목별)
- 경고 사항 및 메모

모달 닫기: 우측 상단 **X 버튼**, 배경 클릭, 또는 **Esc 키**

### 7.4 견적 삭제

목록에서 **휴지통 아이콘** 클릭 후 확인 대화상자에서 승인하면 삭제됩니다. 삭제는 되돌릴 수 없습니다.

### 7.5 CSV 내보내기

이력 화면 우측 상단의 **Export CSV** 버튼을 클릭하면 현재 필터 조건에 맞는 견적 목록이 CSV 파일로 다운로드됩니다.

## 8. 자주 묻는 질문

### Q. 청구중량(Billable Weight)이란?

운송사는 실중량과 부피중량 중 **큰 값**을 기준으로 요금을 부과합니다.
- 부피중량 = (가로 x 세로 x 높이) / 5,000 (UPS/DHL) 또는 / 6,000 (E-MAX)
- 예: 50x40x30cm 박스 = 부피중량 12kg → 실중량 5kg이면 12kg 기준 과금

### Q. Surge / AHS Fee는 언제 입력하나요?

아래 경우에 운송사에서 할증료가 부과될 수 있습니다:
- 단일 박스 중량 25kg 초과 (AHS Weight)
- 가장 긴 변 122cm 초과 (AHS Dimension)
- 둘레(2H+2W+L) 300cm 초과 (Large Package)
- 목상자/스키드 포장 (AHS Packing)
- 성수기 할증 (Peak Season)

정확한 금액은 운송사에 확인 후 해당 필드에 수동 입력하세요.

### Q. E-MAX가 선택이 안됩니다.

E-MAX는 중국(CN)과 베트남(VN)만 지원합니다. 도착 국가를 중국 또는 베트남으로 선택하면 활성화됩니다.

### Q. FSC(유류할증료)는 어디서 확인하나요?

Financial Factors 섹션의 외부 링크를 이용하세요:
- **UPS FSC**: UPS 공식 유류할증료 페이지
- **DHL FSC**: DHL 공식 유류할증료 페이지
- **Naver 환율**: Naver 실시간 환욜조회 및 환율계산기 페이지

FSC는 매주/매월 변동되므로, 견적 작성 시 최신 값을 확인하고 입력해 주세요.

### Q. 견적을 수정할 수 있나요?

저장된 견적은 직접 수정할 수 없습니다. 수정이 필요한 경우:
1. Calculator 탭에서 조건을 변경하여 새로 계산
2. 새 견적으로 저장 (새 참조번호 부여)
3. 필요 시 이전 견적 삭제

### Q. 다크 모드는 저장되나요?

현재 버전에서 다크 모드 설정은 세션 동안만 유지됩니다. 브라우저를 닫으면 기본(라이트) 모드로 돌아갑니다.

### Q. PDF에 참조번호가 "DRAFT"로 나옵니다.

견적을 저장하기 전에 PDF를 다운로드하면 참조번호가 "DRAFT"로 표시됩니다. 견적을 먼저 저장하면 `SQ-YYYY-NNNN` 형식의 실제 참조번호가 표시됩니다.
