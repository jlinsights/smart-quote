## Publishing workflow

1. Open Intercom → Articles → New article
2. Copy the markdown content into the editor (Intercom supports markdown
   via paste, or use the formatting toolbar)
3. Set the audience:
   - Partner-facing articles → "All users" or visitor-facing
   - Admin articles → restrict to admin segment
4. Add to a collection: `Getting Started`, `Quoting`, `Admin`, etc.
5. Publish — Intercom Messenger will surface relevant articles based
   on the partner's current page and recent events

## Update policy

When the product changes in a way that affects these articles (new
carrier, new add-on, UI rewrite), update the markdown here first, then
re-publish in Intercom. Keeping the source in-repo lets us diff changes
against product releases.
