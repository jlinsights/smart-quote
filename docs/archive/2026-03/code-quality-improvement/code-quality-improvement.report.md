## 6. Lessons Learned

1. **단일 소스 원칙**: 환율/FSC 같은 자주 변경되는 값은 반드시 하나의 상수 파일에서 관리
2. **클라이언트 보안**: 이메일/ID 같은 식별자를 프론트엔드 코드에 하드코딩하지 않기 — 역할 기반 체크 사용
3. **데드 코드 관리**: `hidden` CSS로 숨긴 코드는 결국 데드 코드 — 즉시 제거하거나 feature flag 사용
